# entwine - Enterprise Digital Twin

LLM-powered simulation of SME operations. Models employee actions and interactions with digital platforms using AI agents.

## Vision

Create a digital twin of a small-to-medium enterprise (up to ~12 employees) where LLM agents enact employee roles, interact with external platforms (X, LinkedIn, Gmail/Office365, Reddit, etc.), and leverage RAG for domain knowledge.

## Status

**All milestones complete.** Core agent runtime, platform integrations (Slack, GitHub, Gmail, X, LinkedIn-simulated), cost tracking with budget enforcement, end-to-end scenario tests, and performance benchmarks are implemented. See [`docs/project-summary.md`](docs/project-summary.md) for details.
